# 🎓 Bachelor's Thesis
### Control System for Cooling, Condensation, and Water Recirculation
**Platform:** Emerson Ovation 4 | **Institution:** Technical University of Cluj-Napoca (2026)
*   Modeled and implemented a full control system for a Rankine-cycle process, focusing on condenser pressure, condensate level, and recirculation flow control.
*   Designed Process and Instrumentation Diagrams (P&IDs), PI controllers, automatic start/stop sequences, redundancy mechanisms, interlock logic, and complete operator graphics.
*   Validated the system through open-loop identification, comparison with MATLAB models, real-time logic verification, and rigorous fault-scenario testing.

# 🚀 Projects Portfolio

## ⚙️ Process Control & Industrial Automation
*   **Automated Level Control System (Festo Process Control Station):** Modeled and controlled the liquid level of a two-tank laboratory station using a PLC-based control loop (**Siemens Simatic S7-300**). Derived a nonlinear Simulink model from first principles and identified a linear first-order model from step-response data. Tuned PI controllers using pole-placement and Nichols-chart approaches, and validated feedforward and cascade control structures for disturbance rejection.
*   **Boiler-Turbine-Generator Control System:** Designed principal automatic control loops for a radial power system (live-steam pressure, drum water level, active power, frequency regulation). Derived FOPDT transfer function models and tuned PI/PID/PD controllers using the modulus criterion and Ziegler-Nichols method. Validated loops via **Simulink** simulation.
*   **Self-Balancing Two-Wheeled Transporter:** Developed cascade control logic to stabilize an unstable transporter and track a reference trajectory. Identified 12 linear transfer functions and a 10-state MIMO model; designed inner-loop PID (tilt) and outer-loop PI (position) controllers via the imposed phase-margin method.
